# 🎵 Music Controller

O **Music Controller** é um projeto de alta performance projetado para gerenciar áudio e automação de mídia via WebSockets e entrada direta de teclado.

Utilizando o `uv` da [Astral](https://astral.sh/uv), este repositório é otimizado para velocidade de desenvolvimento e gerenciamento de dependências.

## 🚀 Como Começar

### Pré-requisitos
- [uv](https://docs.astral.sh/uv/getting-started/installation/) instalado no seu sistema.

### Instalação
Clone o repositório e sincronize o ambiente:
```bash
uv sync
```

### Execução
Para rodar o controlador principal:
```bash
uv run main.py
```

## 🛠️ Tecnologias
- **Python 3.13+**: O que há de mais recente no ecossistema.
- **WebSockets**: Para comunicação em tempo real.
- **Keyboard**: Para captura e emissão de eventos globais de teclado.
- **Ruff & MyPy**: Garantindo qualidade de código e segurança de tipos.

## 📐 Estrutura do Projeto
- `main.py`: Ponto de entrada (a ser expandido).
- `pyproject.toml`: Configuração central de dependências.
- `.cursorrules`: Regras de IA para desenvolvimento colaborativo.
